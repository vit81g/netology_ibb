1. готовим файл Dockerfile
Содержание Dockerfile

# качаем образ scratch
FROM scratch

# копируем в образ файл app.bin
COPY app.bin /

# при запуске контейнера выполниться команда /app.bin
CMD ["/app.bin"]

2. подготовим app.bin
chmod +x app.bin

3. Собером образ 
docker build -t <имя образа>
Примерный результат :

4. покажет наши образы 
docker image ls

5. запустит наш образ по имени образа
docker container run <имя_образа>
