MITRE ATT&CK
Матрица векторов атак
Threat hunting (TH)

Описывает цели, техники и тактики атакующих (для TH нужно только 12 тактик)

MITRE ATT&CK не является исчерпывающей
MITRE ATT&CK можно использовать с нуля для построения процесса TH

Тактики MITRE ATT&CK
1. Initial access       - методы изначального доступа к сети
2. Execution            - методы исполнения вредоносного кода
3. Persistence          - методы организации постоянного доступа к хосту / сети
4. Privilege escalation - методы повышения привилегий
5. Defence evasion      - методы обхода детектирования
6. Credential access    - методы получения доступа к учетным данным пользователей
7. Discovery            - методы разведки ресурсов внутри скомпрометированной сети
8. Lateral movement     - методы продвижения внутри скомпрометированной сети
9. Collection           - методы сбора информации внутри скомпрометированной сети
10. Command and control - методы подключения и взаимодействия со скомпрометированными хостами
11. Exfiltration        - методы кражи данных
12. Impact              - методы иного воздействия на инфраструктуру

У каждой тактики есть подтехники атак

1. Initial access:
Подтехника Phishing (Фишинг)
- T1566.001 Spearphishing Attachment  (фишинг с вложением)
- T1566.002 Spearphishing Link        (фишинг с сылкой)
- T1566.003 Spearphishing via Service (фишинг через сервис, обратная связь на сайте)

- T1566.001 Spearphishing Attachment (целевой фишинг с вложением) атакующий использует
письмо с вложением для получения доступа. Задача - доставка полезной нагрузки.
Приведены примеры разных хакерских групп, разного ВПО

Важно:
MITRE ATT&CK показывает какие тактики и техники используют атакующие. Позволяет
делать предположения о действиях каких то конкретных атакующих, о действиях связанных
с хакерской групировкой
