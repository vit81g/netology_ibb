Sysmon - компонент Windows Sysinternals
простота установки
дополнительные логи
сохраняет логи в стандартной папке windows\system32\winevt\logs

sysmon.exe -i accepteula -h md5, sha256 -l -n

-h - какие хеши логировать
-l - логировать загрузку модулей
-n - логировать сетевые соединения

-l -n важные элементы

можно запускать с конфигурационными файлами. Берем на Github
