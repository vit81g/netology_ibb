Задание 3
1.    Определите, какой режим доступа установлен для файлов /etc/passwd и /etc/shadow.
2.    Объясните, зачем понадобилось именно два файла.

1.1 Для passwd установлен ограниченный режим доступа: владелец - root с правами чтение запись, группа - root с правами чтение, для всех остальных только чтение.
1.2 Для shadow установлен ограниченный режим доступа: владелец - root с правами чтение запись, группа - root с правами чтение, для всех остальных доступ закрыт. 
Данные файлы используются для аутентификации входа в систему. Файл /etc/passwd представляет собой текстовый файл с одной записью в строке, представляющей учетную запись пользователя (id, группы, домашний каталог) . /etc/shadow — это текстовый файл, содержащий информацию о паролях пользователей системы. Он принадлежит пользователю root и группе shadow и некому не доступен, так как содержит информацию о паролях пользователей в зашифрованном виде.
